# Bettr UI Example

This is an example project showing how to use Bettr UI components in a Next.js application.

## Getting Started

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Open http://localhost:3002
```

## Features Demonstrated

- ✅ Glass Card component with variants
- ✅ Gradient buttons
- ✅ Icon circles
- ✅ Feature badges
- ✅ Clean mode cards
- ✅ Dark mode support
- ✅ Responsive design

## Project Structure

```
bettr-ui-example/
```
