CoFounder Documentation

Welcome to the CoFounder documentation. CoFounder is the AI guardrail layer for every agent framework — providing production-ready security, compliance, and cost optimization for AI applications.

What is CoFounder?

CoFounder is a comprehensive TypeScript-first framework with 35+ packages that provides:

  • Runtime guardrails — PII detection, prompt injection blocking, toxicity filtering
  • 9 compliance frameworks — HIPAA, GDPR, CCPA, SEC, PCI, FERPA, SOX, Safety, Enterprise
  • Agent SDK wrappers — Guardrails for Anthropic Agent SDK, OpenClaw, LangChain, CrewAI
  • Cost optimization — 70% savings through caching, model routing, and budget enforcement
  • Observability — Real-time dashboards, Prometheus export, anomaly detection
  • CI/CD security — GitHub Actions, pre-commit hooks, SARIF output
Tip: CoFounder works with any LLM provider — Anthropic, OpenAI, Google, AWS Bedrock, and more.

Installation

Install CoFounder packages via npm:

# Core guardrails
npm install @aicofounder/guard

# Agent SDK wrapper (for Anthropic Agent SDK)
npm install @aicofounder/agent-sdk

# Compliance enforcement
npm install @aicofounder/compliance

# All-in-one (guard + compliance + policies + agent-sdk)
npm install @aicofounder/guard @aicofounder/compliance @aicofounder/policies @aicofounder/agent-sdk

Requirements

  • Node.js 18.x or higher
  • TypeScript 5.3+ (recommended)
  • npm, yarn, or pnpm package manager

Quick Start

Get guardrails running in under 5 minutes:

Step 1: Install the Guard Package

npm install @aicofounder/guard

Step 2: Add Guardrails to Your LLM Calls

import { createGuard } from '@aicofounder/guard';

const guard = createGuard({
  pii: true,        // Detect and redact PII
  injection: true,  // Block prompt injection
  cost: true,       // Track costs
  budget: 10.00     // $10 budget limit
});

// Use like your normal LLM client
const response = await guard.chat('Summarize this document...');
console.log(response.text);
console.log(response.cost); // $0.003

Step 3: Add Compliance (Optional)

import { ComplianceEnforcer, PresetRules } from '@aicofounder/compliance';

const enforcer = new ComplianceEnforcer([
  PresetRules.hipaa(),
  PresetRules.gdpr()
]);

// Validate before sending to LLM
const result = enforcer.validate(userInput);
if (!result.passed) {
  console.log('Compliance violation:', result.violations);
}
Note: CoFounder wraps your existing LLM calls — no need to change your agent architecture.

@aicofounder/guard

Lightweight, zero-dependency runtime guardrails. Wraps any LLM API with PII detection, injection blocking, cost tracking, and rate limiting.

import { createGuard } from '@aicofounder/guard';

const guard = createGuard({
  pii: true,
  injection: true,
  toxicity: true,
  cost: true,
  rateLimit: { maxRequests: 100, windowMs: 60000 }
});

@aicofounder/agent-sdk

Guardrail wrapper for the Anthropic Agent SDK. Provides 7 composable interceptors and pre-built compliant agent factories.

import { createGuardedAgent, AgentFactories } from '@aicofounder/agent-sdk';

// Pre-built HIPAA-compliant agent
const agent = AgentFactories.hipaa({
  model: 'claude-sonnet-4-20250514',
  tools: [myTool]
});

// Or compose your own interceptors
const agent = createGuardedAgent({
  interceptors: ['pii', 'injection', 'cost', 'audit']
});

@aicofounder/compliance

Enterprise compliance enforcement with 9 built-in presets.

import { ComplianceEnforcer, PresetRules } from '@aicofounder/compliance';

const enforcer = new ComplianceEnforcer([
  PresetRules.hipaa(),   // Healthcare
  PresetRules.gdpr(),    // EU data protection
  PresetRules.pci(),     // Payment card industry
  PresetRules.sox(),     // Financial reporting
]);

@aicofounder/policies

Declarative policy engine with YAML-based compliance presets and conflict resolution.

import { PolicyEngine } from '@aicofounder/policies';

const engine = new PolicyEngine({
  policies: './policies/',  // Load from YAML files
  strict: true
});

CoFounder detects 20+ PII patterns including SSN, credit cards, emails, phone numbers, addresses, and more. PII is automatically redacted before data reaches LLMs.

Prompt Injection Blocking

25+ attack patterns are detected and blocked in real-time, protecting your agents from adversarial inputs.

Cost Optimization

Real-time cost tracking across 25+ LLM models, intelligent caching, model routing, and budget enforcement deliver up to 70% cost savings.

Observability Dashboard

The @aicofounder/dashboard package provides real-time monitoring for costs, security events, compliance scores, and anomaly detection with Prometheus export.

import { Dashboard } from '@aicofounder/dashboard';

const dashboard = new Dashboard({
  port: 3001,
  prometheus: true

CI/CD Security

The @aicofounder/ci package provides GitHub Actions and pre-commit hooks for scanning PRs.

# .github/workflows/cofounder.yml
- uses: waymaker-ai/cofounder-action@v1
  with:
    scan: 'pii,injection,secrets,models'
    fail-on: 'high'
    format: 'sarif'

Troubleshooting

Common Issues

Guard not detecting PII

Ensure PII detection is enabled in your guard configuration:

const guard = createGuard({ pii: true });

Cost tracking not working

Set your provider API keys so CoFounder can map models to pricing:

const guard = createGuard({
  cost: true,
  providers: { anthropic: process.env.ANTHROPIC_API_KEY }
});
