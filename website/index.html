How It Works

1

Install Packages

Add CoFounder guardrails to your project. Choose the packages you need — from lightweight runtime guards to full compliance enforcement.

npm install @aicofounder/guard @aicofounder/compliance @aicofounder/agent-sdk
2

Wrap Your AI Agents

CoFounder wraps your existing LLM calls with guardrails. No need to change your agent architecture — just wrap and go.

import { createGuard } from '@aicofounder/guard';
const guard = createGuard({ pii: true, injection: true, cost: true });
3

Enforce Compliance

Apply compliance presets for your industry. CoFounder automatically validates inputs and outputs against regulatory requirements.

import { ComplianceEnforcer, PresetRules } from '@aicofounder/compliance';
const enforcer = new ComplianceEnforcer([PresetRules.hipaa()]);
4

Monitor & Optimize

Track costs, security events, and compliance scores in real-time with the observability dashboard.

import { Dashboard } from '@aicofounder/dashboard';
const dashboard = new Dashboard({ port: 3001 });

Before vs After CoFounder

// Raw LLM call - no guardrails

With CoFounder
