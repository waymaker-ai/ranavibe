# Gradient Button

Beautiful button component with gradient backgrounds and multiple variants.

## Installation

```bash
npx @bettr/ui-cli add gradient-button
```

## Usage

```tsx
import { GradientButton } from '@/components/ui/gradient-button'

export default function Example() {
  return (
    <GradientButton variant="purple">
      Click me
    </GradientButton>
  )
}
```

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'purple' \| 'orange' \| 'blue' \| 'green' \| 'outline' \| 'ghost'` | `'purple'` | Button style variant |
| `size` | `'sm' \| 'md' \| 'lg' \| 'xl' \| 'icon'` | `'md'` | Button size |
| `animation` | `'none' \| 'pulse' \| 'bounce' \| 'glow'` | `'none'` | Animation effect |
| `disabled` | `boolean` | `false` | Disable button |
