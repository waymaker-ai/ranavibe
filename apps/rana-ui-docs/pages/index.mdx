## Installation

```bash
npm install @rana/ui class-variance-authority clsx tailwind-merge
```

Or use the CLI:

```bash
npx @rana/ui-cli init
npx @rana/ui-cli add glass-card
```

## Quick Start

```tsx
import { GlassCard, GradientButton } from '@rana/ui'

export default function App() {
  return (
    <GlassCard variant="tinted" padding="lg">
      <h1>Welcome to RANA UI</h1>
      <GradientButton variant="purple">
        Get Started
      </GradientButton>
    </GlassCard>
  )
}
```

## Components

- [Glass Card](/components/glass-card) - Frosted glass card component
- [Gradient Button](/components/gradient-button) - Button with gradient backgrounds
- [Icon Circle](/components/icon-circle) - Circular icon container
- [Feature Badge](/components/feature-badge) - Badge for highlighting features
- [Clean Mode Card](/components/clean-mode-card) - Minimal card component

## CLI

RANA UI comes with a powerful CLI for managing components:

```bash
# Initialize
rana-ui init

# Add components
rana-ui add glass-card gradient-button

# List all components
rana-ui list

# Add all components
rana-ui add --all
```
